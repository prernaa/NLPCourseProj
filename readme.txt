Data Processing

Preliminary experimentation folder:
This contains the source code and data used for preliminary experiments on a subset of the data without feature selection.

Supervised and unsupervised folder:
This contains the source code and data for the final model and results generated on the entire dataset.

Data Statistics folder:
Contains statsMeasure.py file used for computing the statistics for the corpora (training + validation + test data). Statistics include average number of words per tweet, ratio of opinionated to neutral tweets, ratio of positive to negative tweets

youtube video presentation link.txt:
Contains the link for the video demonstration

Video presentation slides.ppt:
It contains the powerpoint slides explaining different aspects of the project. The same slides have been used for video demonstration.
